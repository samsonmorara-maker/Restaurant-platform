#  TastyBites
Is a web application that allows customers to discover restaurants, see they menus,reserve tables and pre-order meals
## Features
### Search and filter restaurants
Allows users to quickly find restaurants based on their restaurant name,Food cuisine and keywords input
### View restaurants and thier menus
See available restaurants and open the restaurants page where you find its menu ,view prices and select meals
### book table and pre-order meals in advance
Allows users to choose booking date and time also reserver table in  advance
## Tech Stack
- React
- React Router
- Tailwind CSS
- vite
- Json Server
- Firebase authentication
- Github and render for deployment
## How it works
TastyBites allows customers to browse and search resturants open restaurant detail pages view menus select meals
for pre-order, reserve tables  also allowing restaurant owners with visibility.When a customer books a table the platform 
collects  details such as date, time number of guests and selected meals,then sends the booking data to a database using json 
server.
## Future Improvements
- Real backend APIs
- Real time notifications
- Chat system between restaurants and customers
- AI recomendations
- Checkout system
